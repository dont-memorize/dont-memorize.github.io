# Madhavi Tutoring Website

A simple static website for Madhavi Tutoring. The page advertises tutoring services, subject offerings, testimonials, and includes a zcal embed for booking a demo session.

## Files

- `index.html` contains the page markup and zcal booking embed.
- `styles.css` contains all page styling.
- `assets/tutoring-hero.png` is the hero image used on the landing banner.

## Running Locally

Open `index.html` directly in a browser. No build step or local server is required.

## Notes

The booking section uses zcal's inline widget, so the live scheduler requires an internet connection.
